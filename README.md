# Pneumonia Detection AI

This project is a Deep Learning application designed to identify pneumonia from chest X-ray images. By utilizing a Convolutional Neural Network (CNN), the system can classify images into **Normal** or **Pneumonia** categories to assist in medical image analysis.

## 🚀 Live Demo
The application is currently deployed and can be accessed at:  
**[pneumonia-detection-ai-production.up.railway.app](https://pneumonia-detection-ai-production.up.railway.app)**

---

## 📌 Project Overview
The goal of this project is to provide an automated tool for preliminary X-ray screening. This can be especially useful in reducing the workload of healthcare professionals and providing quick insights into lung health.

### Key Features
* **Automated Detection:** Uses a trained CNN model for image classification.
* **Web-Based Interface:** Simple UI for users to upload X-ray images and get instant results.
* **Cloud Hosted:** Easily accessible via the web through the Railway deployment.
